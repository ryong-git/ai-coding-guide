# 전사 활용 가능한 AI 코딩 가이드 제안

## 🎯 핵심 변경 사항

**기존:** MSP 엔지니어 중심 (AWS, 인프라, 코딩)
**변경:** 전 직군 활용 가능 (영업, 마케팅, HR, 기획, 재무 등)

### 왜 중요한가?
- AI 도구는 **코딩 전용이 아님**
- 모든 부서가 문서 작성, 데이터 분석, 자동화 필요
- 회사 전체의 생산성 향상 = 더 큰 ROI
- 부서 간 AI 활용 문화 확산

## ⚡ Quick Win #1: 긴 문서 요약하기 (5분)

### 누구에게 유용한가?
✅ 영업: 고객사 RFP 문서 요약
✅ 마케팅: 경쟁사 백서 분석
✅ HR: 채용 관련 법규 요약
✅ 기획: 시장 조사 보고서 요약
✅ 엔지니어: 기술 문서 요약

### 실습: 10페이지 문서를 3줄로 요약

**1단계: 문서 준비 (1분)**
- PDF, Word, 웹페이지 등 어떤 형식이든 OK
- 텍스트를 복사할 수 있으면 됨

**2단계: AI에게 요청 (1분)**
```
다음 문서를 3가지 관점에서 요약해줘:

[문서 내용 붙여넣기]

1. 핵심 메시지 (3줄)
2. 주요 내용 (5개 bullet points)
3. 액션 아이템 (우리가 해야 할 것)
```

**3단계: 결과 활용 (3분)**
- 요약본을 팀에 공유
- 회의 자료로 활용
- 의사결정 참고

### 💡 실제 예시

**입력:** 20페이지 AWS 백서
**출력:**
```
핵심 메시지:
1. 클라우드 마이그레이션은 단순 이전이 아닌 비즈니스 혁신 기회
2. 6R 전략(Rehost, Replatform, Refactor 등)을 상황에 맞게 선택
3. 평균 27%의 비용 절감과 3배의 운영 효율 향상 가능

주요 내용:
• 마이그레이션 준비 단계: 현황 분석 및 비즈니스 케이스 수립
• 6가지 마이그레이션 전략과 각각의 장단점
• 보안 및 컴플라이언스 고려사항
• 성공 사례: 금융, 제조, 유통 업계
• ROI 측정 방법 및 KPI 설정

액션 아이템:
• 현재 인프라 현황 조사 시작
• 파일럿 프로젝트 후보 선정
• 예산 및 일정 초안 작성
```

### ✅ 체크포인트
- [ ] 문서를 AI에 입력함
- [ ] 요약 결과를 받음
- [ ] 실제 문서와 비교해서 정확한지 확인
- [ ] 팀원 1명에게 공유

### 실습: 고객 미팅 후속 이메일

**1단계: 상황 정리 (2분)**
```

포함 내용: 미팅 요약, 다음 단계, 일정 제안
```

**2단계: AI에게 요청 (1분)**
```
다음 상황에 맞는 이메일을 작성해줘:

받는 사람: A사 김부장님 (IT 담당 임원)
상황: 어제 클라우드 마이그레이션 관련 미팅 진행
목적: 
- 미팅 감사 인사
- 논의 내용 간단 요약
- 제안서 발송 예정 안내
- 다음 미팅 일정 제안

톤: 정중하고 전문적이지만 너무 딱딱하지 않게
길이: 200-300자 정도

추가 정보:
- 논의 주제: 온프레미스 → AWS 마이그레이션
- 예상 기간: 3개월
- 다음 미팅 제안: 다음 주 화요일 또는 수요일
```

**3단계: 검토 및 발송 (2분)**
- 생성된 이메일 검토
- 필요시 수정
- 발송

### 누구에게 유용한가?

```
회의명: Q1 마케팅 전략 회의

메모:

- 타겟: 중소기업 IT 담당자
- 예산: 5천만원
- 채널: LinkedIn, 블로그, 웨비나
- 이대리: 콘텐츠 제작 담당
- 박사원: 웨비나 기획
- 다음 회의: 2주 후

다음 형식으로:
1. 회의 개요
2. 주요 논의 사항
3. 결정 사항
4. 액션 아이템 (담당자, 기한)
```

## 📊 부서별 상세 가이드 (Part 0.3)

### 영업팀을 위한 AI

#### Use Case 1: 제안서 초안 작성
**상황:** 고객사 RFP에 대한 제안서 작성

**프롬프트:**
```
당신은 베스핀글로벌의 시니어 세일즈입니다.
다음 RFP에 대한 제안서 초안을 작성해주세요:

고객사: A금융사
요구사항:
- 온프레미스 → AWS 마이그레이션
- 현재 서버 50대
- 예산: 5억원
- 기간: 6개월
- 보안 및 컴플라이언스 중요

다음 구조로:
1. Executive Summary
2. 제안 배경 및 목적
3. 제안 내용 (기술 접근 방법)
4. 프로젝트 일정
5. 예상 비용
6. 기대 효과
7. 베스핀글로벌 강점

각 섹션은 2-3 문단으로 작성.
```

**예상 시간 절감:** 3시간 → 30분 (83%)

#### Use Case 2: 고객 미팅 준비
**상황:** 내일 고객사 미팅, 준비 자료 필요

**프롬프트:**
```
내일 B제조사와 클라우드 도입 관련 첫 미팅이 있습니다.
다음 정보로 미팅 준비 자료를 만들어주세요:

고객사 정보:
- 업종: 제조업 (자동차 부품)
- 규모: 직원 500명
- 현황: 레거시 시스템 노후화
- 관심사: 비용 절감, 생산성 향상

준비 자료:
1. 미팅 아젠다 (30분)
2. 질문 리스트 (현황 파악용)
3. 참고할 유사 사례 (제조업)
```

## 📊 ROI 계산기

### 개인 레벨
```
월간 절감 시간 = (작업 시간 × 절감률 × 빈도) × 4주

예시 (영업팀):
- 제안서: 8시간 × 75% × 2회 × 4주 = 48시간/월
- 이메일: 0.3시간 × 60% × 10회 × 4주 = 7.2시간/월
- 회의록: 0.5시간 × 80% × 5회 × 4주 = 8시간/월

총 절감: 63.2시간/월 (약 8일)
```

### 팀 레벨
```
팀 절감 시간 = 개인 절감 × 팀원 수

예시 (영업팀 10명):
63.2시간 × 10명 = 632시간/월 (약 80일)

금액 환산 (시급 5만원 기준):
632시간 × 5만원 = 3,160만원/월
```
